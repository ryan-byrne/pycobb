# pycobb

## Installation

### Using PIP

```
pip install pycobb
```

### From Source

```
git clone https://github.com/ryan-byrne/pycobb
cd pycobb
python install setup.py
```

## Usage

### From the Command Line

`pycobb` is utilized most easily from the command line. For example:

```
pycobb get -p Trevor+Bauer -b Jose+Abreu+0 --print
```
